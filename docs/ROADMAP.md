# Project Roadmap

Near-term:

- Bandwidth source abstraction.
- Optional Npcap detection and permission UX.
- Router counter sources.
- Per-device bandwidth attribution.
- Bandwidth dashboard and reports.

Later:

- Quotas and safe bandwidth limits.
- Presence history and family profiles.
- Privacy mode and CVE/CPE correlation.
- Local API hardening.
- Optional agent collector.
- Accessibility, localization, and release-candidate polish.
