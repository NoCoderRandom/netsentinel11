# Privacy Review

NetSentinel11 is local-first. Inventory, labels, workspaces, reports, and API
tokens are intended to stay on the local Windows machine unless the operator
explicitly exports them.

Default privacy controls:

- Local REST API disabled by default.
- Local API binds to loopback and requires a token when enabled.
- Reports are generated locally.
- Packet capture/Npcap is optional and must be operator-approved.
- No cloud account or subscription lock-in is required.
